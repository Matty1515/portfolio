Hi, I'm Matt an aspiring Front-End developer from Bristol, UK. I specialise in creating imaginative websites and responsive UI, using new frameworks and tools to push boundaries and materialise great designs. Scroll down to see what I have already made.

Websites

Visit Tweethorpe Coastline

One page website advertising the beautiful Tweethorpe coastline. Like many of my websites, it features a print inspired layout with details about the local wildlife, walking routes and attractions.

Hamilton's Tennis Club

A one-page site for Hamilton's Tennis Club. It advertises the skills that you can gain by being part of a tennis club in a responsive, grid based layout. The site uses git, npm and parcel.js under the hood.

ReactJS Calculator

A calculator web app built with ReactJS. User input from buttons is stored in the state, concatenated to create an equation, and then evaluated when the 'equals' sign is clicked. Calculator design is sculpted with CSS grid syntax to create a modern responsive layout.

Matsuko Landscaping

A landing page for the landscaping and gardening services company Matsuko Landscaping. Sass and CSS grid are used together to craft a print-like layout that focuses on telling it's brand's story through engaging text and images.

Kube Design Agency

The landing page for the website design agency Kube. ScrollMagic and GSAP are combined to create complex animations that animate on scroll. Styles are created with SCSS and modern CSS grid syntax. ParcelJS is the bundler and it is deployed with Netlify.

English Rose Rugby

An ongoing project of mine is the website englishroserugby.com. It is a mobile-first project with a high emphasis on usability and responsive design. Images are modified with Gimp and webpages optimised with SEO best practices.
