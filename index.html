Websites

ReactJS Calculator

A calculator web app built with ReactJS. User input from buttons is stored in the state, concatenated to create an equation, and then evaluated when the 'equals' sign is clicked. Calculator design is sculpted with CSS grid syntax to create a modern responsive layout.

Buddha Quotes

A ReactJS web app for finding a random quote from the Buddha! Quotes are stored in an array, displayed with JSX and can be generated at random by clicking 'next'. There is also a Twitter button which uses the Twitter API to tweet the the quote.

English Rose Rugby

An ongoing project of mine is the website englishroserugby.com. It is a mobile-first project with a high emphasis on usability and responsive design. Images are modified with Gimp and webpages optimised with SEO best practices.

Travel Site Mockup

A website for finding useful travel information (weather forecast, popular places, etc.) about a selected city. Using modern fetch() syntax, the site makes requests to two external APIs (Foursquare and APIXU) and then renders the received JSON data into HTML.
