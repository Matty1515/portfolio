info

The website also includes helpful information about formulae and usage.

Travel Site Mockup

info

A website for finding useful travel information (weather forecast, popular places, etc.) about a selected city. Using modern fetch() syntax, the site makes requests to two external APIs (Foursquare and APIXU) and then renders the received JSON data into HTML. The site makes use of Sass to keep a clean and consistent style-sheet.

ReactJS Calculator

info

A calculator web app built with ReactJS. User input from buttons is stored in the state, concatenated to create an equation, and then evaluated when the 'equals' sign is clicked. Calculator design is sculpted with CSS grid syntax to create a modern responsive layout.
